# Dashboard Service Utilisateur — titres de sections harmonisés

Cette mise à jour harmonise les titres des quatre sections avec le style de l’indicateur « SUJETS EN COURS » :

- texte en capitales ;
- couleur grise ;
- corps compact ;
- même graisse et même espacement des lettres.

Sections concernées :

- RTU ;
- RÉSEAU, RACCORDEMENT ET FLUX ;
- EMM ;
- CHANTIERS DE CONCEPTION.

## Publication

Déposer `index.html` et `.nojekyll` à la racine du dépôt GitHub Pages. Pour forcer le rafraîchissement dans Grist, ajouter temporairement un paramètre d’URL, par exemple `?v=1.8`.
